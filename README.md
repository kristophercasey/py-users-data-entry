# Template ETL

## Stack

- python 3.9
- conda 4.9.2
- pip 21.0.1

#### Setup local environment and plugins
- conda create --name name_env

- conda activate name_env

- conda install pip

- pip install -r requirements.txt

#### Run application

python ./main.py


#### Repository workflow

- ALWAYS new features into new branch

- ALWAYS update local changes using "git pull --rebase"
